Software Concepts

Software can be defined as instructions or procedures that hardware performs in order to achieve a desire outcome, and can be further classified into two categories, Application software and System software.


System Software

System software is used in the operation of a computer. Dealing with all types of Operating system tasks, it communicates directly to system hardware. System software can be further classified into either System Management software or system development software. The former is used in the maintenance and operation of a computer, the whole the former is used to create software, whether for the system or an application.



Application software

Application software can be generalized as a set of instructions for a computer to accept a certain input and performs operations to achieve a certain output. Application software can also be further classified into 2 different categories, General-purpose and application specific software. General-Purpose software can be described as a suite of tools for a user to be able to accomplish a task, while an application specific software is designed to fulfill a specific function.
